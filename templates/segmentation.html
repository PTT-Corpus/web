Segmentation

Jseg 結巴斷詞系統 Github repo
  • - An enhanced version of Jieba segmentator.
  • - Using directed acyclic graph (DAG) and Hidden Markov Model (HMM) to segmentate Chinese words.
  • - Data are trained with Sinica Corpora. Equipped with Emoticon detection. F1-score: 0.91
PyCCS 中研院線上斷詞API Github repo
  • - An API for CKIP Segmentator written in Python
  • - Automatically separate word and POS tags
  • - Html-like tag can be segmentated normally
Segcom
  • - A tool for comparing the different results from Jseg and PyCCS
